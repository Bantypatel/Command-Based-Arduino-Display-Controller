# Command-Based Arduino Display Controller

This Arduino project displays messages on a 16x2 LCD based on user commands sent via Serial Monitor.

## Features
- Displays "HELLO WORLD" and custom text commands
- Serial-controlled interactive display
- Easy to expand for IoT or automation use

## Components
- Arduino Uno
- 16x2 LCD display
- Potentiometer (10kÎ©)
- Jumper wires

## Commands
| Command | Action |
|----------|--------|
| HELLO | Displays "HELLO WORLD!" |
| WELCOME | Displays "WELCOME USER!" |
| CLEAR | Clears the display |
